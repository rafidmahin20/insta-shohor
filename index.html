how does javascript works?

javascript is a synchronus single threaded language. It means js can execute one command at a time in a specific order that means it can only go to the next line when the current line is executed. js is lightweight, cross platform and object oriented programming language. js doesn't take much time to load in a machine. And using few techniques js can execute faster than other languages.

Difference between local storage & session storage

local storage has size of 10mbps but sometimes it shows 5mb if you search deeply u can get that the size is 10mb. And session storage has 5mbps size. in local storage u have to delete the data by yourself but in session when u close the window or tab it automatically clear the storage. Local storage works for any tab. Whereas in session storage only works for specific or particular tabs when u close the tab it also clears the storage. local storage and session storage stores data in client side.
